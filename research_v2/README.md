# AI Resume Matcher

AI Resume Matcher — система интеллектуального подбора кандидатов на вакансии, которая анализирует резюме и вакансии, извлекает ключевые навыки и ранжирует кандидатов по релевантности. Проект использует NLP-модели для эмбеддингов текста и метрики NDCG и Spearman для оценки качества ранжирования.


## Основные возможности

* Извлечение ключевых навыков из резюме и вакансий с помощью NLP.
* Ранжирование резюме для каждой вакансии на основе совпадений навыков.
* Метрики качества:

  * NDCG — оценивает порядок рекомендаций по сравнению с экспертными аннотациями.
  * Spearman — корреляция между предсказанным и реальным порядком.
  * Поддержка DOCX и CSV форматов для резюме и вакансий.

## Использование

### Запуск обучения и оценки модели
'''
python -m src.train_and_evaluate
'''
* Сценарий автоматически:

  * Загружает резюме и вакансии.
  * Извлекает ключевые навыки.
  * Ранжирует резюме для каждой вакансии.
  * Выводит метрики NDCG и Spearman.
  * Проводит K-fold кросс-валидацию (по умолчанию 5 фолдов).

## Цель проекта

Помочь HR-специалистам быстро оценивать кандидатов и автоматически определять наиболее релевантные резюме по каждой вакансии, с прозрачной метрикой качества подбора.
