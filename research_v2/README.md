# AI Resume Matcher

Проект в разработке для хакатона. Проект помогает проверять резюме на соотвествие вакансии . 

## Описание 

Ассистент:

- Извлекает навыки из резюме и вакансий (KeyBERT+ Sentense-transformers)
- Сравнивает навыки и ранжирует резюме по соответствию вакансии
- Вычисляет метрики качества:
  - NDCG
  - Spearman correlation
- Имеет веб-интерфейс на Streamlit:
  - загрузка резюме (.docx)
  - выбор вакансии
  - вывод процента соответствия навыков
