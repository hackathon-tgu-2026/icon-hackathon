Комплексная система для автоматического матчинга резюме с вакансиями на основе векторного поиска и комбинированного скоринга. Использует модель nomic-embed-text-v1.5 для анализа семантического сходства и встроенное извлечение ключевых навыков.
Основной функционал:
Векторный поиск: кодирование текстов резюме и вакансий в многомерное векторное пространство.
Комбинированный скоринг: учёт косинусного сходства (60%), перекрытия навыков (25%) и соответствия по длине текста (15%).
Оценка качества: расчёт метрики NDCG@5 на основе экспертных аннотаций.
ИИ-ассистент для HR: генерация персонализированных рекомендаций вакансий.
Визуализация: 4-панельные графики с анализом распределения метрик.

VacancyResumeMatcher

AIAssistant
HR-ассистент для генерации отчётов.
Скоринговая система
Метрика NDCG@5
Нормализованная кумулятивная усиленная выгода учитывает порядок вакансий в ground truth.
Формат входных данных:
CSV вакансий (5_vacancies.csv;)
Резюме (CV/*.docx):
Стандартные документы Word с текстом резюме в параграфах.
Ground Truth:
Функция load_ground_truth() содержит экспертные аннотации в формате:
Ключ: ID резюме (1-30)
Значение: список ID вакансий в порядке релевантности
Визуализация (evaluation_results.png)
4 панели:
График NDCG по резюме — линейная динамика с средним значением;
Гистограмма распределения — сколько резюме в каждом диапазоне NDCG;
Статистика и качество — основные метрики и итоговая оценка;
Кумулятивное распределение — какая доля резюме имеет NDCG ≥ X.
